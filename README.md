# ChatBot_System
디지털 스마트 부산 6기 최종 6조 질의 응답 시스템 개발 Repository입니다 
개인에게 맞춤화된 취업 정보, 도움을 제공합니다.
이력서를 기준으로 사용자에게 취업 공고를 추천해주고, 원한다면 추가적인 조언, 정보를 제공해줍니다.

## 로컬 실행 위해 설치
```
pip install -r requirements.txt
```

## 가상 환경 실행
```
# Window
python -m venv venv
venv\Scripts\activate
```
```
# macOS
python -m venv venv
source venv/bin/activate
```

## 개발 환경
- Python 3.12
- Front : Html, CSS ,JavaScript
- BackEnd : Django
